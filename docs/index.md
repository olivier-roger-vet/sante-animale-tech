# Memento de Santé Animale

!!! note "Usage et public du site"

    Ce site web constitue un aide mémoire de santé animale destiné aux technicien(ne)s de laboratoire d'analyse vétérinaire. Vous y trouverez des fiches techniques contenant l'essentiel de ce qu'il faut savoir sur les principales affections touchant les animaux d'élevage, pour lesquelles le recours au laboratoire d'analyse est fréquemment requis : 

    - Les [maladies infectieuses](./maladies_infectieuses/index.md)
    - Les [maladies parasitaires](./maladies_parasitaires/index.md)

    Bonne lecture !

    Cette documentation est volontairement simplifiée, et n'a donc pas vocation à être exhaustive.
